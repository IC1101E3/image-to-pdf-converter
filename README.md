# image-to-pdf-converter

## Описание проекта

Этот проект представляет собой кросс-платформенный сервис для конвертации изображений в форматах JPG, JPEG и PNG в PDF-файлы. Приложение разработано на языке Python с использованием графического интерфейса на базе PyQt5, что обеспечивает удобный пользовательский опыт. Сервис поддерживает логгирование операций, валидацию входных файлов и unit-тестирование для обеспечения надёжности.

Ключевые особенности:
- **Кросс-платформенность**: Работает на Windows, Linux и macOS без дополнительных модификаций.
- **Графический интерфейс**: Простой GUI для выбора изображений, указания пути сохранения PDF, нового имени файла и выполнения конвертации.
- **Обработка ошибок**: Валидация форматов изображений, обработка исключений с логгированием.
- **Логгирование**: Запись событий в файл и консоль с ротацией логов для предотвращения переполнения.
- **Тестирование**: Unit-тесты с использованием pytest для проверки ключевых функций.
- **Соответствие стандартам**: Код соответствует PEP 8, использует строгую типизацию (typing), docstrings в стиле Google. Комментарии в коде — на русском языке для удобства русскоязычных разработчиков.

Проект модульный, с разделением на бизнес-логику (конвертация), GUI и утилиты (логгирование). Это делает его легко расширяемым, например, для добавления новых форматов или функций.

## Требования

- Python 3.8 или выше (рекомендуется 3.12 для оптимальной производительности).
- Виртуальное окружение (venv) для изоляции зависимостей.
- Установленные зависимости из `requirements.txt`.

## Установка

1. **Клонируйте репозиторий**:
   ```
   git clone https://github.com/IC1101E3/image-to-pdf-converter.git
   cd image-to-pdf-converter
   ```

2. **Создайте виртуальное окружение**:
   - На Linux/macOS:
     ```
     python -m venv venv
     source venv/bin/activate
     ```
   - На Windows:
     ```
     python -m venv venv
     venv\Scripts\activate
     ```

3. **Установите зависимости**:
   ```
   pip install -r requirements.txt
   ```

   Зависимости включают:
   - `pyqt5==5.15.10` — для графического интерфейса.
   - `pillow==10.0.0` — для обработки изображений.
   - `img2pdf==0.4.4` — для конвертации изображений в PDF.
   - `pytest==7.4.0` — для тестирования.

## Запуск приложения

Для запуска приложения из корневой директории проекта используйте команду, чтобы правильно обработать пакетную структуру:
```
python -m src.main
```

Это откроет графическое окно, где вы сможете:
- Выбрать одно или несколько изображений (JPG, JPEG, PNG).
- Указать путь для сохранения PDF, новое имя файла (автоматически добавляется расширение `.pdf`, если оно отсутствует).
- Выполнить конвертацию с отображением результата (успех или ошибка).

Примечание: Если вы запускаете из директории `src/`, используйте `python main.py`, но рекомендуется запуск из корня для избежания проблем с импортами.

## Тестирование

Проект включает unit-тесты для модуля конвертации. Для запуска тестов:
```
pytest tests/
```

Тесты проверяют валидацию изображений и процесс конвертации с использованием временных файлов. Убедитесь, что виртуальное окружение активировано и Pillow установлен для создания тестовых изображений.

## Логгирование

Логи сохраняются в директории `logs/` (создаётся автоматически при первом запуске). Файл лога — `converter.log` (или другой, указанный в `setup_logger`). Логгирование включает:
- Уровень INFO по умолчанию.
- Вывод в консоль и файл.
- Ротацию файлов (максимум 5 МБ, 3 резервных копии).

Пример записи в лог: информация о выбранных файлах, ошибках открытия изображений или успешной конвертации.

## Разработка и стиль кода

- **Стиль**: Код форматирован в соответствии с PEP 8. Рекомендуется использовать инструменты вроде Black для автоматического форматирования и Flake8 для линтинга.
- **Типизация**: Используется модуль `typing` для аннотаций типов, что улучшает читаемость и позволяет проверку с Mypy.
- **Комментарии**: Все комментарии и docstrings на русском языке для удобства.
- **Расширение**: Для добавления новых функций (например, поддержки других форматов) модифицируйте `converter.py` и добавьте тесты.
